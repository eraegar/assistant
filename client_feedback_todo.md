# Client Feedback - To-Do List

## High Priority Features

### 1. Multiple Assistant Assignment System
- **Objective**: Enable the assignment of multiple assistants to a single client.
- **Instructions**:
  1. Develop a system that allows multiple assistants to be assigned to one client.
  2. Create functionality for reassigning assistants as needed.
  3. Implement a method for distributing tasks among the assigned assistants.
  4. Ensure there is a process for replacing assistants during their vacation or sick leave.
- **Business Logic**: 
  - Allow multiple assistants per client.
  - Enable dynamic reassignment of assistants.
  - Manage tasks efficiently among multiple assistants.

### 2. Task Filtering System
- **Objective**: Introduce comprehensive filtering options for tasks.
- **Instructions**:
  1. Add filters to sort tasks by their status, such as in progress, completed, or awaiting revision.
  2. Implement date-based filtering to organize tasks by the date they were received.
  3. Allow filtering by specific managers to view tasks under their supervision.
  4. Enable filtering by assistant to see tasks assigned to them.
  5. Integrate a search function to find tasks based on their content.
- **UI**: Incorporate filter controls into the task management interface for easy access.

### 3. Subscription Management Enhancements
- **Objective**: Improve tracking and notifications for client subscriptions.
- **Instructions**:
  1. Add a column labeled "End Date" to the client list to display when subscriptions expire.
  2. Enable filtering of clients based on their subscription end dates.
- **UI**: Display the subscription end date in the manager dashboard for better visibility.

### 4. Tariff Plan Display
- **Objective**: Simplify the display of tariff plans with clear options.
- **Instructions**:
  1. Present three main tariff categories: Personal, Business, and Full.
  2. For each category, offer three options for hours: 2 hours, 5 hours, and 8 hours.
  3. Calculate prices based on the selected tariff type and hours.
  4. Develop a dynamic pricing system to adjust prices accordingly.
- **Business Logic**:
  - Personal tariff: Offers basic personal assistance.
  - Business tariff: Provides business-focused assistance.
  - Full tariff: Combines personal and business assistance.
  - Hour options: Choose from 2, 5, or 8 hours of dedicated assistant time.
  - Pricing: Base tariff price plus the cost of selected hours.
- **UI**: Design a clean and simple interface for selecting tariffs and viewing prices.

## Medium Priority Features

### 5. Telegram Usage Statistics
- **Objective**: Monitor and analyze client interactions with the Telegram bot.
- **Instructions**:
  1. Track how users engage with the Telegram bot.
  2. Measure conversion rates from bot interactions to registered clients.
  3. Create reports detailing user activity and engagement levels.
- **Business Value**: Gain insights into client behavior and identify areas for improvement.
